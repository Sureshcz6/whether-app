# 🌦 Weather App (Ultra UI)

A modern and responsive weather web application that shows real-time weather data, dynamic background videos, and live map based on the searched city or state.

---

## 🚀 Features

- 🌍 Search weather by city or state
- 🌡 Real-time temperature, humidity, and weather condition
- 🎥 Dynamic background video based on weather (Rain, Clouds, Clear)
- 🗺 Live map integration using location coordinates
- ⌨️ Enter key support for fast search
- 📱 Fully responsive (Mobile + Desktop)
- 💬 Feedback section
- 🔗 Social media icons and affiliate link section

---

## 🛠 Technologies Used

- HTML5
- CSS3 (Glassmorphism UI)
- JavaScript (Fetch API)
- OpenWeatherMap API
- Leaflet.js (Map integration)
